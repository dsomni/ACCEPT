Условные обозначения

[#] - значение можно изменить в системных настройках

[+] - редактируемое значение

2. Главная страница

На главной странице располагаются несколько[#] новостей в сокращённом виде: заголовок новости[+], краткое описание[+] и дата создания. Снизу располагается кнопка "Перейти ко всем новостям", которая перенаправляет на страницу со всеми новостями в том же сокращённом виде. В нижней части страницы со всеми новостями, находится меню навигации по страницам, которое формируется автоматически в зависимости от количества новостей на странице[#] и их общего количества.

Чтобы раскрыть содержимое новости, щёлкните по её названию.

Чтобы добавить новость, кликните на ссылку "Добавить новость" на любой из упомянутых страниц. Интерфейс добавления новости идентичен интерфейс редактирования, который будет рассмотрен ниже.

2.1 Страница новости

Новость состоит из (сверху вниз): изображения[+], названия[+], панели управления, автора, даты и времени создания, содержимого[+].

Для перехода к редактированию нажмите на кнопку [редактировать] в панели управления. Аналогично нажмите на кнопку [удалить] для удаления.

2.3 Добавление и редактирование новости

Меню добавления (редактирования) содержит следующие поля:

 1. Заголовок
  2. Описание: краткое описание новости, которое будет отображено на главной странице
  3. Текст новости в HTML формате. Это означает, что можно использовать html теги, css стили и javascript скрипты.

>>>>>
  4. Изображение. Является опциональным полем, будет отображено в самом верху новости. Для вставки изображений внутри новости, рекомендуем использовать тег img
